## 1. 用户相关接口

### 1.1 注册

#### 1.1.1 基本信息

> 接口名称：register
>
> 身份限制：无
>
> 接口描述：该接口用于注册添加用户

#### 1.1.2 请求参数

请求参数说明：

| 参数名称 | 说明 | 类型 | 是否必须 | 备注 |
| ---- | -------- | ------ | ------ | ----------|
| name | 用户名   | String | 是    |          |
| password | 密码  | String | 是   | 加密     |
| role | 用户身份  | String | 是    | 值必须为Administrator, Browser或Operator |

#### 1.1.3 响应数据

响应参数说明：

| 名称 | 类型 | 是否必须 | 默认值 | 备注 | 其他信息 |
|----|----|----|----|----|----|
| code | number | 是 | | 响应码, 0-成功,1-失败 |  |
| message | String | 否 | | 提示信息 |  |
| data | Object | 否 | | 返回值 |  |
